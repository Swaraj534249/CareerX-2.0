# MERN_Bank_crud-app

## Procedure to run

1. Open project in IDE

2. Open terminal
Run below commands in order

`npm install`
`nodemon app.js`

this will run ru your server

3. Open another terminal

`cd ./bank/`
`npm install`
`npm start`
